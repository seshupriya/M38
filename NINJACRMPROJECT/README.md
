# Selenium Java Automation Framework

## Project Overview
This project is a Selenium-based automation testing framework developed to automate a live web application.
The framework is built using Java, Selenium WebDriver, TestNG, Maven, and follows the Page Object Model (POM)
design pattern.

The framework focuses on automating business-critical workflows and supports regression execution.

---

## Tech Stack
- Programming Language: Java
- Automation Tool: Selenium WebDriver
- Test Framework: TestNG
- Build Tool: Maven
- Design Pattern: Page Object Model (POM)
- Version Control: Git & GitHub

---

## Framework Structure
- pages: Contains page classes with locators and business actions
- tests: Contains TestNG test classes
- utils: Utility classes for WebDriver, Excel, properties, and common reusable methods
- base: Base class for browser setup and teardown

---

## Application Under Test
A live web application used for automation practice.
